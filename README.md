# 🎨 Magic Camperbot Profile

这是我的第一个完整的HTML/CSS/JavaScript项目！基于freeCodeCamp的调试练习扩展而成。

## ✨ 功能特点
- ✅ 响应式个人资料页面
- ✅ 点击页面任意位置，背景颜色随机变化
- ✅ 精心设计的文字阴影效果

## 🖱️ 如何使用
1. 打开 `Camperbot_Profile.html`
2. **点击页面任意位置** - 背景会随机变色！
3. 多次点击体验不同颜色

## 🛠️ 技术栈
- HTML5 - 页面结构
- CSS3 - 样式设计  
- JavaScript (ES6) - 交互逻辑
